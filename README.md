DESCRIPCIÓN
================================================================
 
Juego Conecta 4 digital para dos jugadores. El objetivo es
alinear 4 fichas del mismo color en horizontal, vertical
o diagonal en un tablero de 7 columnas x 6 filas.
 
================================================================
FUNCIONALIDADES
================================================================
 
- 4 interfaces gráficas: Inicio, Registro, Juego y Resultados
- Animación de caída de fichas
- Cronómetro por ronda
- Marcador acumulado entre rondas
- Detección automática de ganador y empate
- Animación de confeti al ganar
 
================================================================
CÓMO EJECUTAR
================================================================
 
1. Abrir el proyecto en Apache NetBeans
2. Verificar que "connec44.jpg" esté en la carpeta src/
3. Ejecutar Inicio.java como clase principal
4. Ingresar nombres de jugadores y presionar REGISTRAR
5. Hacer clic en una columna del tablero para jugar
